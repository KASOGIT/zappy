Detailed Description

TabButton is used in conjunction with a TabBar.


  TabBar {
      TabButton {
          text: qsTr("Home")
      }
      TabButton {
          text: qsTr("Discover")
      }
      TabButton {
          text: qsTr("Activity")
      }
  }

Note: Types in the Qt.labs module are not guaranteed to remain compatible in future versions.
